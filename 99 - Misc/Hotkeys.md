## Поиск

- Ctrl + F = Поиск
	- + Shift = Поиск в текущем файле
- Alt + Q = Текущий файл в менеджере файлов
	- + Ctrl = Поиск по папке в менеджере файлов
- Alt + S = Поиск в файле(Омни)
	- + Ctrl = Поиск везде(Омни)

## Создание файлов

- Ctrl + N = Новый файл
- Ctrl + Shift + N = Создать холст

## Управление файлами
- Ctrl + W = Закрыть файл
- Alt + <- = Назад
- Alt + -> = Вперед

## Шаблоны
- Alt + C = Блок кода
- Alt + D = Разделитель
- Alt + T = Таблица
